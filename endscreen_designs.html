GeoCatch — End Screen Redesign

Combined analysis from VECTOR (Game Designer) and UI/UX Visionary

Both Experts Agree

Both the game designer and UI/UX expert independently recommend Concept B — "Creature Showcase / Creature Parade" as the strongest end screen design. It puts caught creatures front-and-center, replaces negative "GAME OVER" language with celebratory messaging, and gives the family-friendly GeoCatch identity a proper send-off.

VECTOR additionally recommends borrowing the fireworks particle effect from Concept E for new high-score moments only — a lightweight enhancement that doesn't add complexity to the normal flow.

Shared Design Principles (Both Experts)

  • Show the trainer sprite prominently — players identify with their character
  • Replace "GAME OVER" / "TIME'S UP" with positive language ("Great Run!", "Expedition Complete!")
  • Make caught creatures the hero of the screen — they're the reward
  • Always show navigation hints (R to Restart, Q to Quit) — no dead-end traps
  • Cap score count-up animation at ~2 seconds max regardless of score value
  • Name entry should be clearly visible and inviting, not hidden at the bottom
  • High score celebration should be distinct but not overwhelming for a family game

Design Concepts

Concept A — "Expedition Debrief" / "Trophy Ceremony"

Both Experts

Theme: A formal results screen with trophy/medal imagery. Score displayed prominently with a stats breakdown panel showing creatures caught, accuracy, and streaks.

Pros

  • Clean, readable layout
  • Stats breakdown adds replay value
  • Familiar "results screen" pattern

Cons

  • Doesn't showcase the creatures themselves
  • Can feel clinical / generic
  • Trophy metaphor doesn't match creature-catching theme

Sources: VECTOR "Trophy Ceremony" + UI/UX "Expedition Debrief"

Concept B — "Creature Showcase" / "Creature Parade"

★ RECOMMENDED

Theme: The creatures you caught are the stars. All 5 creature stickers displayed in a row — caught ones in full color with a gentle bob animation, uncaught ones greyed out with "?" overlay. Trainer sprite shown alongside. Positive tier-based message replaces "GAME OVER".

Tier Messages (based on score thresholds):

  • 5/5 caught → "LEGENDARY EXPLORER!"
  • 4/5 caught → "AMAZING RUN!"
  • 2-3 caught → "GREAT JOB!"
  • 0-1 caught → "NICE TRY!"

VECTOR's parade twist: Creatures march across the screen with a gentle bob animation, sky-blue/grass-green background reminiscent of the game world. On new high score, add fireworks particle burst (borrowed from Concept E).

A/B Variant — Show All 5 vs Only Caught

Variant A (Recommended): Show all 5 creature stickers. Caught = full color + checkmark. Uncaught = greyed silhouette + "?". This creates a "gotta catch 'em all" motivation for replay.

Variant B: Only show caught creatures. Cleaner but removes the collection-completion incentive.

Pros

  • Creatures are the hero — matches game identity perfectly
  • Grey silhouettes create replay motivation
  • Positive messaging feels rewarding for kids
  • Two-column bottom layout is clean and functional
  • Trainer sprite adds personal connection

Cons

  • More implementation work than simpler designs
  • Need to handle variable caught counts gracefully

Sources: VECTOR "Creature Parade" (Rank 1) + UI/UX "Creature Showcase" (Rank 1)

Concept C — "Scoreboard Podium" / "Family Hall of Fame"

Both Experts

Theme: A large, prominent leaderboard as the centerpiece. Gold/silver/bronze podium styling for top 3. Current run highlighted if it places. Family-oriented — emphasizes competition between players (DAD vs LEO).

┌──────────────────────────────────────────┐ │ ⭐ HALL OF FAME ⭐ │ │ │ │ ┌───┐ │ │ ┌──┤ 1 ├──┐ │ │ ┌──┤ │DAD│ ├──┐ │ │ │ 3│ │2270│ │ 2│ │ │ │LEO│ └───┘ │LEO│ │ │ │355│ │475│ │ │ └───┘ └───┘ │ │ │ │ Your Score: 1280 → NEW #2! 🎉 │ │ │ │ [ Enter Name: ___________ ] │ │ │ │ [R] Play Again [Q] Menu │ └──────────────────────────────────────────┘

Pros

  • Perfect for family competition (DAD vs kids)
  • Podium visual is satisfying when you place
  • Simple and focused layout

Cons

  • Disappointing when you don't place in top 3
  • Doesn't showcase creatures at all
  • Podium rendering is complex for limited payoff

Sources: VECTOR "Family Hall of Fame" + UI/UX "Scoreboard Podium"

Concept D — "Card Reveal" / "Celebration Burst"

Both Experts

Theme: Dramatic reveal animation. Each caught creature gets a "card flip" or burst animation, shown one at a time with a brief delay. Final score revealed after all creatures shown. High-energy, arcade feel.

┌──────────────────────────────────────────┐ │ │ │ ┌────────────────────┐ │ │ │ ✨ FIRE DRAKE ✨ │ ← card flips│ │ │ │ in one by │ │ │ [creature art] │ one with │ │ │ │ 1s delay │ │ │ +320 pts │ │ │ └────────────────────┘ │ │ │ │ After all cards: │ │ TOTAL SCORE: 1280 │ │ [Enter Name] [R] [Q] │ └──────────────────────────────────────────┘

Pros

  • Most dramatic and exciting reveal
  • Each creature gets individual spotlight
  • Builds anticipation for final score

Cons

  • Slow — players may want to skip after first time
  • Complex animation state machine
  • Frustrating if you only caught 1 creature

Sources: VECTOR "Pokemon Card Reveal" + UI/UX "Celebration Burst"

Concept E — "Fireworks Finale" / "Map Postcard"

Both Experts

Theme: A celebratory particle/fireworks display over the game map (or a postcard snapshot of the map). Score floats up with sparkle trails. Most visually impressive but potentially distracting from the actual information.

┌──────────────────────────────────────────┐ │ ✦ ✧ ✦ ✧ │ │ ✧ ✦ SCORE: 1280 ✦ ✧ │ │ ✦ ✧ ✦ ✧ ✦ │ │ │ │ [map background faded] │ │ ┌─ · ─ · ─ · ─ · ─ · ─┐ │ │ · (your trail on map) · │ │ └─ · ─ · ─ · ─ · ─ · ─┘ │ │ │ │ ┌────────────────────────┐ │ │ │ Name: ___ [SAVE] │ │ │ └────────────────────────┘ │ │ [R] Play Again [Q] Menu │ └──────────────────────────────────────────┘

Pros

  • Most visually spectacular
  • Map postcard creates a "memory" feeling
  • Fireworks particle system is reusable

Cons

  • Particles can obscure score/name entry
  • Map rendering on end screen adds complexity
  • Fireworks feel disconnected from creature theme
  • Heavy for every game — better as high-score-only bonus

Sources: VECTOR "Fireworks Finale" + UI/UX "Map Postcard"

Implementation Recommendation

Concept B "Creature Showcase" as the base design, with the following enhancements:

  • All 5 creature stickers in a row (caught = color + ✓, uncaught = grey + "?")
  • Trainer sprite displayed alongside score card
  • Tier-based positive message instead of "GAME OVER"
  • Score count-up animation (capped at 2 seconds)
  • Two-column bottom: leaderboard left, name entry right
  • Fireworks particle burst on NEW HIGH SCORE only (from Concept E)
  • Gentle creature bob animation for caught creatures
  • Always-visible [R] Play Again / [Q] Menu navigation
